This directory lists the files for Tranfer learning approach for the 2D class selection.  The data setup, results generation 
and analysis are described in the technotes.doc (MS word file).

The TL uses the Inceptionv3 model. Details of that model could be found at: https://github.com/tensorflow/models/tree/master/research/inception and at https://cloud.google.com/tpu/docs/inception-v3-advanced
